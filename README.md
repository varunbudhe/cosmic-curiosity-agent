# 🪐 Cosmic Curiosity Agent

Cosmic Curiosity Agent is an AI-powered educational agent that answers universe and space “what if” questions using scientific reasoning, imagination, and structured scoring.

It is built to make complex space science easier, more fun, and more engaging for curious learners.

## 🚀 What it does

Users can ask questions like:

- What if Earth had rings like Saturn?
- What if Jupiter became a star?
- What if the Moon disappeared?
- Can humans live on Titan?
- What happens if you fall into a black hole?

The agent gives:

- Short answer
- Scientific breakdown
- Reality score out of 10,000
- Danger level
- Cinematic imagination scene
- Fun facts
- Final verdict

## 🧠 Why this is an agent

This is not just a random chatbot. It follows a fixed mission and structured behavior:

- It answers only space, universe, physics, planet, alien life, and cosmic “what if” questions.
- It separates real science from imagination.
- It gives a structured output every time.
- It uses scoring to evaluate scientific possibility.
- It explains complex topics in simple language.
